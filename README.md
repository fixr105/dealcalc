# EV Deal Modeller — Seven Fincorp

## Setup

1. Open this folder in Cursor
2. Open the terminal (Ctrl + `)
3. Run:

```
npm install
npm run dev
```

4. Open http://localhost:5173 in your browser

## Tabs

- **Current Math** — Money Multiplier model for OEM pitch (Stage 1)
- **Leasing Model** — 3-phase revenue model (Stage 2)
- **OEM Sale** — Direct B2B model (to be finalised)

All inputs are editable live. SF margin is highlighted in amber.
